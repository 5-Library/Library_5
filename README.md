### Git Commit Convention
```
[컨벤션명] 작업내용

ex) [Feat] Create 작업
ex) [Style] createPage / updatePage Css변경
ex) [Fix] Delete 메서드 호출 시 CORS 에러 수정
```
<br>

1. Git Convention
  - `[Fix]` : 버그, 오류 해결
  - `[Add]` : Feat 이외의 부수적인 코드 추가, 라이브러리 추가, 새로운 파일 생성 시
  - `[Style]` : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
  - `[Feat]` : 새로운 기능 구현

<br>

2. Branch 전략
  - `main` : 개발 완료 및 배포를 위한 브랜치
